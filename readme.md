一款专为AI用户打造的Chrome插件，精准追踪AI交互中的等待时长与总使用时间，让每一秒AI使用成本都清晰可见 ✨

中文名称：等时仪表盘 | 英文名称：WaitDash | 定位：轻量、精准、无冗余的AI使用时间统计工具

## 📌 项目简介

作为AI重度使用者，你是否经常疑惑：和AI对话时，一半时间都在等待响应？到底花了多少时间在AI交互上？

WaitDash 应运而生——以极简设计、精准统计为核心，专注记录你与各类AI工具（豆包、元宝、ChatGPT、Claude、国内AI等）的交互全过程，重点追踪「等待AI响应的耗时」，同时统计总使用时长，帮你直观掌握时间分配，优化AI使用效率，拒绝无效等待。

✅ 无多余权限，纯本地运行，仅统计AI交互时间，保护隐私
✅ 轻量不占内存，不影响Chrome运行速度
✅ 极简UI，数据直观，上手即用
✅ 开源免费，可自定义适配更多AI平台

## 🚀 核心功能

- **等待时长精准统计**：核心功能！自动捕捉AI响应等待时间，毫秒级记录，区分「主动输入时间」与「AI等待时间」，不遗漏任何一段延迟

- **总使用时长统计**：按天、按小时汇总AI工具总使用时间，生成简易数据看板，一眼看清每日/每周AI使用频次

- **多AI平台适配**：默认适配主流AI工具，支持自定义添加AI网站，灵活适配你的常用平台

- **数据本地存储**：所有统计数据仅保存在本地浏览器，不上传服务器，完全保护个人隐私

- **极简可视化**：插件图标hover即可查看实时数据，点击进入详情页，数据展示简洁不冗余，无需复杂操作

## 🔧 安装教程（Chrome浏览器）

### 方式1：本地加载已打包插件（适合非开发者）

1. 从本仓库 Releases 下载最新版本的 `WaitDash.zip` 压缩包

2. 打开Chrome浏览器，输入地址 `chrome://extensions/` 进入扩展程序页面

3. 开启页面右上角「开发者模式」（Developer mode）

4. 点击「加载已解压的扩展程序」，选择解压后的 `WaitDash` 文件夹，即可完成安装

### 方式2：源码本地构建（适合开发者）

1. 克隆本仓库：`git clone https://github.com/gudao/waitdash.git`

2. 进入项目目录，安装依赖（若有）：`npm install`（无依赖可跳过）

3. 重复方式1的步骤2-4，加载项目根目录即可

## 💡 使用说明

1. 安装完成后，Chrome浏览器右上角会出现 WaitDash 插件图标（极简Logo）

2. 打开任意AI工具（如ChatGPT），开始交互时，插件会自动启动统计

3. hover插件图标：可快速查看「当前会话等待时长」「今日总使用时长」

4. 点击插件图标：进入详情页，查看每日/每周统计数据、各AI平台使用占比

5. 自定义AI平台：在详情页「设置」中，添加AI网站域名，即可适配更多平台

## ❓ 常见问题（FAQ）

- **Q：插件会收集我的AI对话内容吗？**
A：不会！所有数据仅记录「时间统计信息」，不捕捉、不存储任何对话内容，数据全部保存在本地浏览器，隐私绝对安全。

- **Q：为什么部分AI平台无法统计？**
A：目前默认适配主流AI平台，若无法统计，可在插件详情页「设置」中添加对应AI网站域名，或提交Issue反馈，我会尽快适配。

- **Q：统计的等待时长精准吗？**
A：基于Chrome Extension API监听页面交互，毫秒级统计，误差在100ms内，可满足日常使用、效率统计需求。
